This web interface allows you to interact with the Lumo server running on your system.

Getting Started

  1. Make sure the Lumo server is running (use lumo server:start in your terminal)
  2. Check the server status indicator at the top of this page
  3. Enter a command or question in the input field
  4. Select the appropriate command type from the dropdown
  5. Click "Execute" to send the command to the server

Command Types

  • AI (default) - Natural language queries processed by AI
  • Shell - Execute shell commands on the server
  • Agent - Use Lumo's agent capabilities
  • System Health - Check system health
  • System Report - Generate system reports
  • Help - Get help information
  • Config - Modify configuration
